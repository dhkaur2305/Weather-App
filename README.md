Weather App Application (Full-stack)
# Name : Dharampreet Kaur

This is a weather app application built using Django. The app allows user to:
1. search weather by city name and country name
2. view temperature, wind speed and humidity
3. show next 5-days forecast
4. export data in JSON format
5. view all previous records which include option of delete and update

Features:
Use two weather APIs integration.
Database storage using Django models.
Data export functionality
Clean UI design

# Set up instructions:
1. Install packages:
   pip install requests
   pip install django
2. apply migrations:
   python manage.py makemigrations
   python manage.py migrate
3. run the server:
   python manage.py runserver
